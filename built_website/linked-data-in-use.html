Linked data in use

Date 13 September 2014 Category Blog

Também disponível em Português

Today I have presented a talk in Salvador at Front in Bahia 2014.

Front in Bahia 2014 is a conference from the well known Front In brand, specialized in front-end development.

Linked Data in Use

In this talk I walk through Semantic Web initiatives, like RDF and SPARQL, linked data principles, discuss some implementation and adoption issues and talk about semantic annotation in HTML.

Semantic annotation using the Schema.org vocabulary is demonstrated using both HTML 5 Microdata or JSON-LD input.

Schema.org example

There is a strong highlight in benefits seen in Google search results with Rich Snippets, Actions in Email, and Google Now with real examples like the example below.
